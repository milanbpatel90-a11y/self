Test Instructions for Send Button Functionality:

1. Open `index.html` in your browser
2. Open browser developer tools (F12 or Ctrl+Shift+I)
3. Go to Console tab
4. Run this code to create test ticket:
   ```javascript
   localStorage.clear(); 
   const testTicket = { 
       id: 'TKT-1001', 
       timestamp: new Date().toISOString(), 
       mobileNumber: '1234567890', 
       product: 'Test Product', 
       queryDescription: 'This is a test ticket for email functionality', 
       priority: 'High', 
       category: 'Test', 
       status: 'Open', 
       assignedTrainer: null 
   }; 
   localStorage.setItem('selfTicket_tickets', JSON.stringify([testTicket])); 
   localStorage.setItem('selfTicket_trainers', JSON.stringify([
       { id: 1, name: 'Alice Johnson', status: 'Available', load: 0 }, 
       { id: 2, name: 'Bob Smith', status: 'Available', load: 0 }, 
       { id: 3, name: 'Carol Williams', status: 'Available', load: 0 }
   ])); 
   localStorage.setItem('selfTicket_counter', '1001'); 
   location.reload();
   ```

Testing the Send button:
1. Click "Claim/Assign" on test ticket
2. Enter email address (e.g., test@example.com)
3. Click "Assign Ticket"
4. Email preview modal will appear with Send and Close buttons
5. Click Send button - you should see success message

Expected behavior:
- If EmailJS config is missing: Shows error "EmailJS not configured"
- If EmailJS config is valid: Sends email and closes modal
- Error handling for failed sends
